Smart-Dark
==========

Version: 1.2


Introduction 介绍
==========
Protect your eyes. 
auto convert website style to night style,Almost all website.
compatibility Chrome & Firefox & Safari.

智能夜间模式：保护你的眼睛，自动转换网站样式到夜间模式。
几乎大部分网站都适用，并且兼容 Chrome 、 Firefox、Safari。
